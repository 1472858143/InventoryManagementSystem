# IT-006 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 落地业务升级迭代 B：进货、采购退货、销售、客户退货和统一库存变更入口。

**Architecture:** 先建立 `StockChangeService` 和 `stock_log` 来源字段，再让 purchase / sales 四类单据在各自事务中创建主表、明细并逐条调用库存入口。旧 `InboundService` / `OutboundService` 保持可用，新业务不直接更新 `stock.quantity`。

**Tech Stack:** Spring Boot 3.5、MyBatis、MySQL、JUnit 5、Mockito、Vue 3、Element Plus、Vite。
