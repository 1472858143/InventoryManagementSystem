# Agent 执行提示

## 任务目标

请先阅读本任务目录中的全部文件，再开始后续分析、设计或修改。当前阶段仅完成商品库展示页面优化任务的文档结构建立和现状分析，不进入具体页面设计和代码实现。

## 执行约束

- 不直接修改正式项目书、需求分析和系统设计文档正文。
- 不直接覆盖 `sql/market.sql` 或 `Document/3-项目开发/market.sql`。
- 不修改数据库结构。
- 不新增后端接口。
- 不把库存规则写入前端页面。
- 修改前检查 Git 状态，避免覆盖用户未提交内容。
- 每次变更保持范围小而明确。

## 必读文件

1. 根目录 `AGENTS.md`
2. `Document/3-项目开发/迭代优化/AGENT_WORKFLOW.md`
3. 当前任务 `README.md`
4. 当前任务影响分析文件
5. `Document/开发记录/frontend/前端开发规划.md`

## 当前阶段限制

- 只做任务登记、目录结构建立和影响分析。
- `设计方案.md` 暂不展开具体方案。
- 不修改 `frontend/src/views/product/ProductView.vue`。
- 不修改 `frontend/src/api/product.js`。

## 后续阶段提示

后续进入设计阶段时，应先基于项目书、前端开发规划和当前页面现状，明确商品库展示页面的展示目标、字段范围、交互边界和验收标准，再进入代码实现。

## 输出要求

- 汇报实际创建或修改的文件。
- 汇报验证命令和结果。
- 汇报未完成事项和风险。
- 不把未验证内容表述为已完成。
